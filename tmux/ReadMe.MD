# Tmux
===

#### intro
Configure Your Tmux

#### common commands
---

> $ tmux a                                      # attach the tmux session without session name

> $ tmux ls                                     # list the tmux session without session name

> $ tmux a -t [tmux number]

> $ tmux a -t [session name]

> $ tmux new -s [session_name]
